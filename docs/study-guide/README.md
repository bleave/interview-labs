## Study Strategy

For each topic:
1. Learn the definition
2. Understand why it matters
3. Learn a practical example
4. Practice concise verbal delivery

Focus heavily on:
- retries/idempotency
- distributed systems
- Angular state
- observability
- production troubleshooting
- ecommerce checkout flows
- scalability tradeoffs
- operational judgment
